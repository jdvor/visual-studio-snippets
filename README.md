## Shortcuts

* **disposable** - IDisposable.Dispose class method
* **basedisposable** - IDisposable.Dispose class method with inheritance
* **bros** - basic equatable readonly struct
* **ros** - equatable and comparable readonly struct
* **dto** - DTO class
* **xtest**
* **axtest**
* **xptest**
